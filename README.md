# SQL Chinook Database Analysis

## Overview

This project analyzes the Chinook database using SQL to answer business-related questions about customers, sales, employees, albums, artists, and genres.

## Tools Used

* SQL
* SQLite
* DB Browser for SQLite

## Key Questions

* Which customers spent the most money?
* Which albums generated the highest revenue?
* What are the best-selling genres?
* How do employee sales compare?
* Which customers purchase the most tracks?

## Skills Demonstrated

* JOINs
* Aggregate Functions
* GROUP BY
* Subqueries
* Common Table Expressions (CTEs)
* Window Functions
* Data Analysis

## Project Files

* `queries.sql` – SQL queries used in the analysis
* `Chinook_Project_Report.pdf` – Final project report
* `screenshots/` – Query results and visual outputs
